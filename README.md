```
Requirements:
 * Webserver
   o PHP 5.5 or higher
     * ini setting: memory_limit greater than or equal to 64M
   o MySQL 5.0 or MariaDB 10 and higher
 * Source Dedicated Server
   o MetaMod: Source
   o SourceMod: Greater Than or Equal To 1.7
```
